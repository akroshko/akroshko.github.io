Github repositories for software projects

Some of my personal coding projects:

  • Emacs org-table database – https://github.com/akroshko/emacs-otdb

    Emacs org-mode tables are used as a database to help with application-specific calculations, currently used for cooking recipes and backpacking gear. Has an easily editable central database of nutritional values and food costs, or correspondingly for the weights and costs of backpacking gear.

  • Crypt profiles – https://github.com/akroshko/crypt-profiles

    To be released soon!
    Create profiles containing GnuPG and SSH public/private keypairs, which are readily synced between computers. Each profile only requires a master GnuPG public/private key pair, that has a password stored in the agent. Features include: transparent decryption/editing of files, transparent usage of password-protected SSH public/private keypairs, public key encryption for files, and using public key encryption for full hard-drive backups.

  • Emacs org-table logbook – https://github.com/akroshko/emacs-otlb

    To be released soon!
    Emacs org-mode tables used as an activity logbook, currently for running/walking along with data from mutiple GPS sources (either phone, watch, or handheld). Helps keeps track of the mileage for each particular footwear. Future work will involve being able to better sumarize captured data, as well as effectively use, assess, and repair bad or incomplete data.

  • Emacs grep-replace – https://github.com/akroshko/emacs-grep-replace

    To be released soon!
    Allows system-wide grepping of and replacing of text, from within Emacs. Features include: building an index of candidate paths and filetypes, the option of fixed strings rather than regular expressions, instantly greping/replacing a symbol at point with no additional interaction, and managing grep-mode/compilation windows
